## 1. 简介

信息资讯越来越庞杂的现在，在短时间内获取有效信息的需求越来越强烈。
得益于先进的科学技术，开发者们可以围绕一些先进的浏览器实现一些额外的功能，被称之为 `附加组件` / `扩展程序` / `插件`。有一款名为  ***Stylish*** 的附加组件，可以通过给网页增加额外的层叠样式表（CSS）实现 优化网页内容的展现方式，萃取有效的信息 等，从而提高人们生活工作的效率。

<!--more-->

本网站旨在提供一些常用网站的用户样式，然后再附加组件的帮助下，成功的对一些网站产生<s>*物理化学*</s> 作用。从而达到带给你更良好的上网体验的目的。

####2.2 科学打开样式站（可略过）

自备梯子者略过。**具体方法：**

Windows 用户，以 管理员身份运行  [<i class="hx-download"></i>Notepad++](http://notepad-plus-plus.org/) 之类的编辑器，打开：

```
C:\Windows\System32\drivers\etc\hosts
```

Linux 用户，终端 执行
```
sudo gedit /etc/hosts
```

在 hosts 文件最后添加：

```
72.21.81.109  df6a.https.cdn.softlayer.net
```

然后保存关闭，等 DNS 刷新后就行了。
